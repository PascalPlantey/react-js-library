getEventTarget(elt) → {Object|undefined}

Tries to find a target element out of elt.

Description:

  • Tries to find a target element out of elt. Get a value from element (see below) and check if it supports events or not. Displays a console warning is no event target can be found. Order of tests to get a value: if elt is ...

    • a string, consider it is an element ID or querySelector
    • a useRef object, it's input or the wrapped element
    • in all other cases the elt itself
